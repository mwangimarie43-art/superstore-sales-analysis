# Superstore Sales Performance Analysis

## Project Overview
This project analyzes sales performance data from a US-based 
retail superstore covering 2014 to 2017. The goal is to 
identify revenue trends, profitability drivers, and provide 
actionable business recommendations.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn) — data cleaning and analysis
- Power BI — interactive dashboard
- Jupyter Notebook — analysis environment

## Dataset
- Source: Sample Superstore Dataset (Kaggle)
- 9,994 transactions
- 21 columns covering orders, customers, products and regions

## Key Findings
- Total Revenue: $2.3M | Profit Margin: 12.47%
- Technology is the most profitable category
- Heavy discounts (above 50%) generate average losses of $89 per order
- West region leads in both sales and profit
- Sales grew 51% from 2014 to 2017

## Files in this Repository
- superstore_clean.csv — cleaned dataset
- Superstore_Sales_Analysis.ipynb — Python analysis notebook
- Business_Insights_Report.docx — findings and recommendations
